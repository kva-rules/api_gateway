# Spring Cloud Gateway API Gateway

Production-ready API Gateway with Spring Cloud Gateway, security filters, service discovery, monitoring, CI/CD.

## 🏗️ Architecture

```
Microservices → [Security Filters → Rate Limiting → Logging] → Gateway → [Eureka LB → Services]
                        ↓
              Dashboard Aggregation | Health | Metrics | OpenAPI
                        ↓
                    Kafka Events → Observability
```

**Clean Architecture Layers:**

- **Controllers**: HealthController, MetricsController, DashboardController
- **Filters**: AuthenticationFilter, RoleBasedAccessFilter, RateLimiterFilter, LoggingFilter
- **Config**: SecurityConfig, RouteConfig, FilterConfig, KafkaProducerConfig
- **Utils**: JwtTokenValidator, TraceIdGenerator

## 🔄 Routing

**RouteConfig (Service Discovery):**

```
 /api/auth/** → lb://auth-service
 /api/users/** → lb://user-service
 /api/tickets/** → lb://ticket-service
 /api/rewards/** → lb://reward-service
 /api/knowledge/** → lb://knowledge-service
 /api/notifications/** → lb://notification-service
```

**GatewayConfig (Static fallback):**

```
 /api/users/** → localhost:8081
 /api/orders/** → localhost:8082
```

## 🔐 Security Pipeline

**Filter Chain Order (FilterConfig):**

1. **TraceIdGenerator** → X-Trace-Id
2. **AuthenticationFilter** → JWT validation, X-User-Id/Roles
3. **RoleBasedAccessFilter** → RBAC (ADMIN/ENGINEER/MANAGER)
4. **RateLimiterFilter** → 100/min per user/IP (Caffeine)
5. **LoggingFilter** → Request/Response audit
6. **SecurityHeadersConfig** → CSP/XSS protection

**RBAC Rules:**

- `/api/users/**` → ADMIN
- `/api/rewards/**` → ADMIN | MANAGER
- `/api/tickets/**` → ENGINEER | MANAGER
- `/api/knowledge/**` → ENGINEER

## 🚀 How to Run

```bash
# Development
mvn spring-boot:run

# Production
mvn clean package
java -jar target/api-gateway-0.0.1-SNAPSHOT.jar

# With profiles
mvn spring-boot:run -Dspring.profiles.active=prod
```

## 🐳 Docker Setup

```bash
# Build & run
docker-compose up --build

# Individual

docker run -p 8080:8080 api-gateway
```
